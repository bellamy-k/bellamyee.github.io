## ğŸŒ Deployment to GitHub Pages

This project is configured to be easily deployed to GitHub Pages.

1.  **Push your code to GitHub**
    Create a new repository on GitHub and push this code to it.

2.  **Deploy**
    Run the deployment script:
    ```bash
    npm run deploy
    ```
    This command builds the project and pushes the `dist` folder to a `gh-pages` branch.

3.  **Enable GitHub Pages**
    - Go to your repository settings on GitHub.
    - Navigate to "Pages".
    - Ensure the source is set to `gh-pages` branch.
